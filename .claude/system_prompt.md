# VARA Development Context

You are helping develop VARA, a YouTube analytics tool.

## Key Architecture
- Backend: .NET 10 minimal APIs
- Database: PostgreSQL
- Frontend: Svelte + SvelteKit
- Real-time: SignalR
- LLM: Multi-provider abstraction (Claude, GPT-4o, Groq)

## Current Episode
Check the VARA_Episode_Roadmap.md in /docs for what's being built this month.

## Tech Requirements
- Use .NET 10 with dependency injection
- Entity Framework Core for database
- REST API design with DTOs
- Interfaces for abstraction (IYouTubeClient, ILlmProvider, etc.)
- Structured logging
