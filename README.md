## Tech Stack

- Backend: .NET 10 (C#) + PostgreSQL
- Frontend: Svelte + SvelteKit
- Real-time: SignalR
- Infrastructure: Hetzner + Terraform
- LLM: Multi-provider (OpenAI, Anthropic, Groq)
